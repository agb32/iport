===============================
IPvDeviceAdapter with eBUS SDK
===============================

This sample shows how to use the IPvDeviceAdapter interface class for serial communication
through a Pleora GigE Vision or USB3 Vision device.

1. Introduction

Using the IPvDeviceAdapter interface class, this sample shows how to:
 * Get connect to a device
 * Configure and open a serial port a on a Pleora GigE Vision or USB3 Vision device. 
 * Send and receive data through the serial port of the IP Engine

2. Pre-conditions

This sample assumes that:
 * You have a Pleora GigE Vision or USB3 Vision device connected to a network adapter or an U3V device connected to the USB3 interface.

3. Description

3.1 DeviceSerialPort.cpp

Shows how to use the IPvDeviceAdapter class for serial communication through a Pleora GigE Vision or USB3 Vision device. Please refer to the comments in the source code for information on the methods used in this sample.
